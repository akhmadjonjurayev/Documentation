  Asymmetric shifrlash
Qoidalar:
  * Public key -> encrypt
  * Private key -> decrypt
  * Private key -> sign
  * Public key -> verify

RSA - matematik asosda.
 1. p, q  2 ta tub son olinadi.
 2. n = p * q
    f(n) = (p - 1)(q - 1)
 3. Public expontent tanlanadi. e
 4. Private exponent f(n) (Eyler funsiyasi siyasida) o'zaro tub son tanlanadi.
  d = e ^ -1 mod f(n)
  Public key -> (e, n)
  Private key -> (d, n)
  Encryption c = m ^ e mod(n)
  Decryption m = c ^ d mod(n)

RSA da tub sonlarni topishda Miller-Rabin testidan foydalaniladi. gcd(m, n) != 1 bo'lsa RSA buziladi.
